3 versions of efficient markets hypothesis

3种 有效市场假设

  • weak: 新闻不影响以后的价值, 只影响当前和以前的. 技术可分析的
  • semi-strong 消息会连续的影响价值
  • strong 1+2 还可能会对没有披露的信息产生响应.
