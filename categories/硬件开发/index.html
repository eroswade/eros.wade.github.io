GAP和GATT的差别

GAP: 通用访问协议(Generic Attribute Profile).

官方说明
用于广播(advertising)的协议.
包含信息:

  • RSSI
  • Address
  • Address Type (short or long)
  • Advertising Data:
    • ShortenedLocalName
    • Flags (discoverable? can connent?)
    • UUID
    • ManufacturerSpecificData
