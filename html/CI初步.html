understand ci welcome page

The exact address of welcome page is //localhost/index.php/welcome . The enterance of CI is INDEX.PHP. 

class Welcome extends CI_Controller {
              public function index ()
            {
                         $this ->load -> view( 'welcome_message' ); 
            } 
}

welcome is the class name , and the 'W' must upcase.
cuz of function index ,,we can access WELCOME/INDEX without call the function.
That means the address is mush longer: localhost/index.php/welcome/index
construction of URL:

  1. localhost -- IP
  2. index.php -- access position, can be canceled. Before cancel it, u mast write config/route.php
  3. welcome -- class name
  4. index -- function name
  5. and params.
