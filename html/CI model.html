CI model

第一步 配置数据库

WAMP把apache, mysql, php 整合在一起. 这3个软件是做服务器必须的组成部分.
其中

  1. apache 做为服务器,对外提供80端口的HTTP支持.
  2. mysql 提供数据库支持
  3. php 提供动态网站的HTML解析

CI如果放在WAMP的WWW目录里. 就表示需要把数据库配置更改为WAMP的数据库环境.
数据库配置文件在application/config/database.php 

    $db['default'] = array(
    'dsn'   => '',
    'hostname' => 'localhost',  // 数据库地址
    'username' => 'root',       // mysql 用户名 
    'password' => '',           // mysql 密码 安装好没改的话,是空的
    'database' => 'database_name',  // 调用数据库名  mysql有一默认测试数据库test 把这里改成test
    'dbdriver' => 'mysqli',         // 定义数据库为mysql
    'dbprefix' => '',               // 表默认前缀
    'pconnect' => TRUE,
    'db_debug' => TRUE,
    'cache_on' => FALSE,
    'cachedir' => '',
    'char_set' => 'utf8',
    'dbcollat' => 'utf8_general_ci',
    'swap_pre' => '',
    'encrypt' => FALSE,
    'compress' => FALSE,
    'stricton' => FALSE,
    'failover' => array()
);

可以使用TEST这个database做测试. 在WAMP里的phpmyadmin可以找到test

读取新闻条目

以下代码在phpmyadmin的 SQL里运行 

    CREATE TABLE news (
    id int(11) NOT NULL AUTO_INCREMENT,
    title varchar(128) NOT NULL,
    slug varchar(128) NOT NULL,
    text text NOT NULL,
    PRIMARY KEY (id),
    KEY slug (slug)
);

用来创建表. 记得在运行之前选选择一个数据库(点一下左边栏的`test`)
